# 红楼梦文本分析项目

## 项目概述
本项目对红楼梦文本进行了全面的分析和可视化，包括章节统计、词频分析和社交网络分析三个主要任务。

## 项目结构
```
.
├── 专有词汇词典.txt          # 红楼梦专有词汇词典
├── 停用词.txt              # 中文停用词表
├── 红楼梦文本(UTF-8).txt    # 红楼梦原始文本
├── 红楼梦社交网络权重.csv    # 红楼梦人物社交网络权重数据
├── 红楼梦分析.py           # 分析脚本
├── 章节段落字数统计散点图.png # 章节统计散点图
├── 红楼梦高频词汇词云图.png   # 高频词汇词云图
├── 红楼梦人物社交网络图.png   # 人物社交网络图
├── 分析结果总结.md          # 详细分析结果总结
├── 任务完成报告.md          # 任务完成报告
├── 最终结果展示.md          # 最终结果展示
└── README.md              # 项目说明文档
```

## 任务说明

### 任务1：章节、段落及其字数统计分析
- 对红楼梦文本进行章节分割
- 统计每个章节的段落数和字数
- 生成散点图展示章节分布情况

### 任务2：分词、去除停用词并生成词云图
- 使用jieba分词对文本进行分词处理
- 去除停用词和单字词
- 统计词频，生成词云图展示高频词汇

### 任务3：绘制人物社交网络图
- 读取社交网络权重数据
- 计算新的权重（chapweight / 段落总数）
- 筛选weight > 0.04的数据
- 使用spring_layout算法进行节点布局
- 绘制人物社交网络图

## 运行环境
- Python 3.8+
- 主要依赖库：
  - jieba==0.42.1
  - pandas==1.3.5
  - matplotlib==3.5.3
  - networkx==2.8.4
  - wordcloud==1.8.2.2

## 运行方法
1. 安装依赖库：`pip install -r requirements.txt`
2. 运行分析脚本：`python 红楼梦分析.py`
3. 查看生成的结果文件
