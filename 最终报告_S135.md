# 红楼梦文本分析项目最终报告

## 项目概述
本项目对红楼梦文本进行了全面的分析和可视化，包括章节统计、词频分析和社交网络分析三个主要任务。通过本项目，我们旨在揭示红楼梦的文本结构、人物关系和社会结构。

## 项目目标
1. 对红楼梦文本进行章节、段落及其字数统计分析
2. 对红楼梦文本进行分词、去除停用词并生成词云图
3. 基于社交网络权重数据，绘制红楼梦人物社交网络图

## 项目实施

### 数据准备
- 红楼梦文本(UTF-8).txt：红楼梦原始文本
- 红楼梦社交网络权重.csv：红楼梦人物社交网络权重数据
- 专有词汇词典.txt：红楼梦专有词汇词典
- 停用词.txt：中文停用词表

### 技术选型
- Python 3.8+
- 主要依赖库：jieba（分词）、pandas（数据处理）、matplotlib（绘图）、networkx（社交网络分析）、wordcloud（词云生成）

### 任务执行

#### 任务1：章节、段落及其字数统计分析
- 对红楼梦文本进行章节分割，共提取74个章节
- 统计每个章节的段落数和字数
- 生成散点图展示章节分布情况
- 统计结果：
  - 总章节数：74
  - 总段落数：1938
  - 总字数：502,707

#### 任务2：分词、去除停用词并生成词云图
- 使用jieba分词对文本进行分词处理
- 去除停用词和单字词
- 统计词频，生成词云图展示高频词汇
- 高频词汇前20个：宝玉、一个、袭人、贾母、凤姐、黛玉、众人、两个、姑娘、只见、老太太、宝钗、王夫人、凤姐儿、奶奶、不知、太太、平儿、东西、贾琏

#### 任务3：绘制人物社交网络图
- 读取社交网络权重数据
- 计算新的权重：weight = chapweight / 段落总数
- 筛选weight > 0.04的数据（共13条边）
- 使用spring_layout算法进行节点布局
- 绘制人物社交网络图
- 核心人物：宝玉（与多个重要人物有紧密联系）

## 项目成果

### 可视化结果
1. **章节段落字数统计散点图**：展示了红楼梦各章节的段落数和字数分布情况，直观反映了章节长度的差异
2. **红楼梦高频词汇词云图**：展示了红楼梦文本中出现最多的150个高频词汇，突出了宝玉、袭人、贾母等核心人物
3. **红楼梦人物社交网络图**：展示了红楼梦人物之间的社交关系网络，清晰地呈现了宝玉与贾母、凤姐、袭人等主要人物的紧密联系

### 分析结论
1. **人物重要性**：宝玉是红楼梦的核心人物，与众多重要角色都有密切联系
2. **社交关系**：贾母、凤姐、袭人、王夫人、宝钗是宝玉最主要的社交网络节点
3. **文本特点**：红楼梦文本篇幅庞大，章节结构丰富，人物关系复杂
4. **文化价值**：通过文本分析和社交网络可视化，可以更深入地理解红楼梦的人物关系和社会结构
